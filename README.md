Vue_toDue is a to-do list which uses Vue and Material Design principles to create a clean front-end interface for the end user. This is a small project I made to reinforce the basic principles of Vue, especially coming from a React background this was an interesting basic exploration of structuring and building a simple project in Vue.js.

![Vue_toDue](Vue_toDue.PNG)

See it in action - 

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
